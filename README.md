
## Room Booking and Management System 
This is a Room Booking and Management System where the admin can manage booking and users to browse available rooms, book them, and view their booking history.
# Technology Stack
## Frontend 
JavaScript<br>
Tailwind CSS<br>
Context API<br>
Redux Toolkit <br>
react-toastify<br>
Moment<br>
JWT
## Backend
JavaScript<br>
Express.js<br>
Mongoose <br>
MongoDB<br>
JWT
